🏷️ 项目名称

AgentForge - 智能任务自动化框架

🛠️ 技术栈

Python, OpenAI API, 小米 MiMo, SQLite, MCP 协议

📝 项目描述

基于 Claude Code 架构设计并实现的智能 Agent 框架，采用 Agent Loop 核心循环、工具注册表模式和分层权限控制，支持异步任务规划、持久化记忆和多 Agent 协作，通过 MCP 协议实现外部工具扩展。项目包含 32 个 Python 源文件、4600+ 行代码，涵盖 8 个核心模块。

✨ 技术亮点（6 项）

  • 实现 Agent Loop 核心循环，支持同步/异步双模式和 stop_reason 安全退出
  • 设计工具注册表模式（Tool Registry），通过装饰器 + 注册函数两步即可添加新工具
  • 构建三级权限门控系统（deny / rules / approval），保障工具执行安全
  • 开发 Hook 事件系统，支持 4 种生命周期事件的灵活拦截与扩展
  • 实现基于 SQLite 的持久化记忆系统，支持 3 种上下文压缩策略
  • 通过 MCP 协议实现多传输层（stdio/SSE/StreamableHTTP）的工具发现与调用

📈 量化成果（6 项）

  • 独立完成 32 个 Python 文件、4600+ 行代码的完整框架开发
  • 实现 8 个高内聚低耦合的核心模块，模块间通过清晰接口通信
  • 内置 5 个开箱即用的工具（Shell、文件读写、搜索等）
  • 设计 4 种 Hook 事件覆盖 Agent 全生命周期
  • 支持 3 种 MCP 传输协议，可连接任意 MCP 兼容工具服务器
  • 实现多 Agent 协作系统，包含消息总线、异步邮箱和团队协调器
