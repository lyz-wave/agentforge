💎 核心设计原则

贯穿 AgentForge 始终的五大工程哲学

🎯

单一职责

每个模块只做一件事，做到极致，模块间通过清晰接口通信。

🔌

可扩展性

Hook、Tool、MCP 三层扩展点，新增能力无需修改核心代码。

🛡️

安全优先

权限门控前置，所有敏感操作必须经过显式授权才能执行。

💾

持久化

对话历史、任务状态、记忆数据均可持久存储，断点续行。

🖥️

交互性

CLI 交互、进度反馈、权限确认——人始终在回路中。

Agency 来自模型。Harness 给 Agency 一个落脚点。
